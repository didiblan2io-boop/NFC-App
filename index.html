Обзор

NFC Master — это профессиональный инструмент для управления NFC (Near Field Communication) метками, спроектированный с соблюдением строгих принципов интерфейса Microsoft Fluent Design и Windows 10 Mobile.

Технологии

Приложение построено с использованием современного стека веб-технологий и аппаратных интерфейсов браузера Google Chrome:
• Web NFC API (NDEFReader / NDEFWriter)
• WakeLock API
• Vibration API
• Web Audio API

Возможности

• Чтение и анализ NDEF-сообщений.
• Запись 14+ типов данных (URL, Text, vCard, Wi-Fi, Crypto и др.).
• Поддержка очереди записей (мульти-запись).
• Клонирование структуры меток.
• Форматирование чипов (Empty NDEF).

Отказ от ответственности (APDU)

В связи с ограничениями безопасности (Sandbox) протокола Web NFC, передача низкоуровневых команд (ISO-DEP / APDU) заблокирована браузерами. Установка и снятие паролей (PWD/PACK) эмулируются интерфейсом в демонстрационных целях.

Политика конфиденциальности

Последнее обновление: 11 Мая 2026

1. Локальное хранение данных

NFC Master уважает вашу приватность. Мы не собираем, не передаем и не храним ваши личные данные на внешних серверах. Вся генерируемая информация (журнал сканирований, история записей, UID меток, настройки интерфейса) хранится исключительно локально в памяти вашего устройства с использованием технологии localStorage.

2. Доступ к аппаратному обеспечению (NFC)

Для корректной работы приложению требуется системное разрешение на использование NFC-модуля. Приложение использует этот модуль строго по вашему прямому указанию (при нажатии кнопок «Сканировать» или «Записать»). Данные с физических смарт-карт и чипов считываются локально и отображаются только вам.

3. Запись данных (NDEF)

При записи пользовательских данных (например, контактов, ссылок, Wi-Fi сетей) на NFC-метку, информация передается напрямую с вашего устройства на чип. Разработчики приложения не имеют доступа к содержимому создаваемых вами записей.

4. Буфер обмена

При включении функции «Авто-копирование UID» приложение получает доступ к буферу обмена для автоматического сохранения серийного номера метки. Это действие происходит локально без сетевых запросов.

5. Удаление данных

Вы имеете полный контроль над своими данными. Вы можете в любой момент удалить всю историю сканирований и сбросить настройки до заводских, воспользовавшись соответствующими кнопками в меню «Журнал» и «Настройки».

6. Изменения политики

Мы оставляем за собой право обновлять данную Политику конфиденциальности по мере добавления нового функционала. Все изменения направлены исключительно на повышение уровня безопасности и прозрачности работы приложения.

Exception: APDU

Операция отклонена политикой безопасности браузера.

Web NFC API Sandbox не поддерживает отправку низкоуровневых команд APDU (ISO-DEP), необходимых для управления PWD/PACK аутентификацией на чипах NTAG/Mifare.
