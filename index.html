Exception: APDU

Операция отклонена политикой безопасности браузера.

Web NFC API Sandbox не поддерживает отправку низкоуровневых команд APDU (ISO-DEP), необходимых для управления паролями NTAG/Mifare.

Используйте нативные Android приложения для этой функции.
